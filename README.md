# ⚡ WiFi-Charge: Data Stronger 📱

**WiFi-Charge** হলো একটি উদ্ভাবনী অ্যান্ড্রয়েড অ্যাপ যা ব্যবহারকারীদের ওয়াই-ফাই ব্যবহারের বিনিময়ে 'ক্রেডিট' প্রদান করে। এই ক্রেডিট ব্যবহার করে ব্যবহারকারীরা অ্যাপ-টু-অ্যাপ কলিং ফিচার উপভোগ করতে পারেন।

## 📖 প্রজেক্ট পরিচিতি
WiFi-Charge অ্যাপটি ব্যাকগ্রাউন্ডে ওয়াই-ফাই কানেকশন ট্র্যাক করে। যখন ইউজার এই অ্যাপের ড্যাশবোর্ড থেকে অন্য নির্দিষ্ট ৯টি অ্যাপ (যেমন: ইউটিউব বা অন্যান্য) ব্যবহার করবেন, তখন তাদের ওয়াই-ফাই ডেটা ব্যবহারের উপর ভিত্তি করে সিস্টেমে ক্রেডিট জমা হতে থাকবে। 

## 🚀 প্রধান বৈশিষ্ট্যসমূহ
- **স্মার্ট ট্র্যাকিং:** ব্যাকগ্রাউন্ডে ওয়াই-ফাই ব্যবহারের পরিমাণ পর্যবেক্ষণ।
- **ক্রেডিট ড্যাশবোর্ড:** রিয়েল-টাইম ক্রেডিট আর্নিং আপডেট।
- **অ্যাপ ইন্টিগ্রেশন:** সরাসরি ড্যাশবোর্ড থেকে ইউটিউবসহ অন্যান্য ৯টি অ্যাপ ব্যবহারের সুবিধা।
- **ফ্রি কলিং:** জমানো ক্রেডিট দিয়ে বন্ধুদের সাথে উচ্চমানের ভয়েস কল।

## 💰 ক্রেডিট সিস্টেম (Earning Logic)
ইউজার যখন ড্যাশবোর্ডের মাধ্যমে অন্য অ্যাপ ব্যবহার করবেন:
1. অ্যাপটি ব্যাকগ্রাউন্ডে ওয়াই-ফাই ডেটা সিগন্যাল ট্র্যাক করবে।
2. ডেটা ব্যবহারের সময়ের ওপর ভিত্তি করে ইউজারের অ্যাকাউন্টে ক্রেডিট "চার্জ" বা জমা হবে।
3. এই ক্রেডিটগুলো পরবর্তী সময়ে কল করার জন্য 'কারেন্সি' হিসেবে ব্যবহৃত হবে।

## 📞 অ্যাপ-টু-অ্যাপ কলিং
- **Spending Logic:** ব্যবহারকারী তার জমানো ক্রেডিট খরচ করে কল করতে পারবেন।
- **কানেক্টিভিটি:** কলটি শুধুমাত্র সেই বন্ধুদের করা যাবে যাদের ফোনে **WiFi-Charge** অ্যাপটি ইনস্টল আছে এবং নম্বর সেভ করা আছে।

## 🛠️ টেকনোলজি স্ট্যাক
- **Language:** Kotlin / Java
- **UI Framework:** Jetpack Compose (Modern Glow Effect Design)
- **Database:** Firebase (Google Login & Data Sync)
- **Architecture:** MVVM

## 🔐 প্রয়োজনীয় পারমিশন (Permissions)
অ্যাপটি সঠিকভাবে কাজ করার জন্য নিম্নলিখিত পারমিশনগুলো প্রয়োজন:
* `INTERNET`: গুগল লগইন এবং ক্লাউড ডাটা সিঙ্ক্রোনাইজেশনের জন্য।
* `QUERY_ALL_PACKAGES`: ফোনে নির্দিষ্ট ৯টি অ্যাপ আছে কি না তা যাচাই করতে।
* `ACCESS_WIFI_STATE`: ওয়াই-ফাই স্ট্যাটাস এবং "চার্জিং" ডিটেক্ট করতে।
* `READ_CONTACTS`: কল করার জন্য কন্টাক্ট লিস্ট থেকে বন্ধুদের খুঁজে পেতে।

## 🎨 ডিজাইন গাইডলাইন
এই অ্যাপটি **"Empty Compose Activity"** ব্যবহার করে তৈরি করা হবে। এর ইউজার ইন্টারফেসে থাকবে একটি আধুনিক **Glow Effect**, যা অ্যাপটিকে একটি প্রিমিয়াম এবং টেক-ফোকাসড লুক দেবে।
